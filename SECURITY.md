# Security Policy

## Supported versions

Only the latest published version of GetGrabKit is supported for security fixes.

## Reporting a vulnerability

Please do not open public issues for vulnerabilities.

Report security issues privately by opening a GitHub security advisory in this repository, or by contacting the maintainer directly through repository contact channels.

Include:

- A clear description of the issue
- Reproduction steps or a proof of concept
- Impact assessment
- Suggested mitigation (if known)

We will acknowledge reports and prioritize confirmed vulnerabilities.
