# Contributing to GetGrabKit

## Development setup

1. Install Node.js 18.17+.
2. Run `npm install`.
3. Run `npm start` to launch the TUI locally.

## Pull requests

1. Create a focused branch for one change.
2. Keep changes small and scoped.
3. Describe what changed and why.
4. Include clear reproduction and verification steps.

## Standards

- Follow existing code style and module structure.
- Avoid unrelated refactors in the same PR.
- Keep error handling explicit and user-facing.
